# Personal Finance Tracker

A **web-based finance management system** that allows users to track their income, expenses, and overall financial health. The application features a clean, modern, responsive interface built using **React.js**. Users can add and delete transactions, visualize spending through charts, and view a real-time summary of their financial activities.
